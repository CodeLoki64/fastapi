# FastAPI Roadmap

An interactive visual roadmap for learning FastAPI — from setup to production microservices. Click any topic to see a description and curated free learning resources.

## Deploy to Vercel (one click)

1. Push this folder to a GitHub repository
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import your repo — Vercel auto-detects Next.js
4. Click **Deploy** — done in ~30 seconds

No environment variables needed.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build for production

```bash
npm run build
npm start
```

## Stack

- **Next.js 14** (Pages Router)
- **React 18**
- Zero external UI dependencies — plain CSS-in-JS

## Customize

- Edit topic descriptions and resources in `data/roadmap.js`
- Add new tiers or nodes in the same file
- Colors are in `styles/globals.css`
